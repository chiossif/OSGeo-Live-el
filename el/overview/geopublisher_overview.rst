Geopublisher
============

Electronic Library Manager
~~~~~~~~~~~~~~~~~~~~~~~~~~Διαχειριστής Ηλεκτρονικής Βιβλιοθήκης

Geopublisher is an atlas authoring system which enables easy publication of geo-data, documents, images, videos, and statistics in form of digital multimedia atlases.Το Geopublisher είναι ένα σύστημα το οποίο καθιστά ικανή την εκτύπωση των γεω-δεδομένων, κειμένων, εικόνων, οπτικομαγνητικών εγγραφών και στατιστικών στοιχείων σε μορφή άτλαντα ψηφιακών πολυμέσων These atlases are effectively a minimal, pre-configured, end-user GIS which offers selected functionality only where its meaningful.Οι άτλαντες αυτοί αποτελούν αποτελεσματικά 

Geopublisher provides tools for quality assurance and multilingual meta-data management.Το Geopublisher παρέχει εργαλεία για ποιοτική επιβεβαίωση και πολυγλωσση διαχείριση μεταδεδομένων. No in-depth knowledge of HTML, SLD or XML is required to create an atlas.Δεν χρειάζεται βαθειά γνώση των HTML, SLD ή XML για να δημιουργηθεί ένας άτλας Atlases can be directly published on CD, DVD, memory stick, external hard-drive or the Internet.Οι άτλαντες μπορεί να δημοσιεύονται απευθείας σε CD, DVD, κάρτες μνήμης, εξωτερικούς σκληρούς δίσκους στο διαδίκτυο. The software is platform-independent.Το λογισμικό είναι ανεξάρτητο πλατφόρμας

Geopublisher's unique features make it especially applicable in the fields of international research, participatory GIS, capacity building, and development cooperation.Τα μοναδικά χαρακτηριστικά του Geopublisher το καθιστούν εφαρμόσιμο κυρίως στα πεδία των διεθνών ερευνών, συμμετοχικά γεωγραφικά συστήματα πληδοφοριών, κτισίματος δυνατοτήτων και ανάπτυξης συνεργασιών.

Core Features
-------------
.. image:: images/screenshots/1024x768/geopublisher-overview.png
  :scale: 40 %
  :alt: screenshot of Geopublisher editing an atlas
  :align: rightΣτοίχιση: Δεξιά

* Atlases created with Geopublisher are a simplified end-user GIS with selected functionality available only where its meaningful.
* Hybrid online and offline atlases make geo-data accessible in areas beyond the digital divide.
* Multilingual management of meta-data avoids redundancies and allows for the quick release of updated atlases.
* Integrated WYSIWYG editors for SLD and HTML allow geopublishing without knowledge of SLD, XML or HTML.
* All software is Free and Open-Source Software compliant with OGC standards.
* Geopublisher + QuantumGIS + OpenOffice provide a free Open-Source Geopublishing desktop.
* :doc:`AtlasStyler <atlasstyler_overview>` SLD editor is integrated into Geopublisher.
* Runs on Mac OS, Windows and Linux.
* Styling based on OGC SLD standard.
* Offline atlases run directly from any media - Java does not have to be installed.
* Supports tiled image pyramids for big raster data.
* Fully translated into German, English, Italian and French.


Implemented Standards
---------------------
* OGC Style Layer Descriptor (SLD)
